# Smart E-Commerce Backend Engine

A backend application built using **Core Java, JDBC, MySQL, and Collections Framework** to manage products, customers, shopping carts, orders, and sales analytics.

## 🚀 Features

### Product Module

* Add Product
* View Products
* Search Product by ID
* Update Product
* Delete Product
* Low Stock Alert

### Customer Module

* Register Customer
* View Customers
* Search Customer
* Update Customer
* Delete Customer

### Cart Module

* Add Product to Cart
* View Cart
* Calculate Total Amount
* Remove Product from Cart

### Order Module

* Place Order
* Order History
* View Order Details
* Search Order by ID
* Update Product Inventory
* Clear Cart After Checkout

### Analytics Module

* Total Revenue Report
* Low Stock Products
* Product Sorting using Collections Framework
